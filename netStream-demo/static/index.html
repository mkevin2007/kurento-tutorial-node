NetStream Demo

This web application consists of a one to many video call using WebRTC. In other words, this is an implementation of a video broadcasting application. There are two types of users in this application: 1 peer sending media (let's call it Presenter) and N peers receiving the media of the Presenter (let's call them Viewers). Thus, the Media Pipeline is composed by 1+N interconnected WebRtcEndpoints. To run this demo follow these steps:

  1. Open this page with a browser compliant with WebRTC (Chrome, Firefox, Opera).
  2. If you would like to be the Presenter of the communication, enter your name and click on Present button. If a presenter with a given name is already present, the application returns an error (see the the console).
  3. Grant the access to the camera and microphone. After the SDP negotiation the communication should start.
  4. In a different machine (or a different tab in the same browser), enter the name of the presenter you want to view and click on View button to see the Presenter's stream. If the stream fails to load then click Stop and click View again.
  5. Click on Stop or close the screen to finish the communication. When the Presenter clicks on Stop or closes his screen, the rest of Viewers (if any) ends the communication too.
